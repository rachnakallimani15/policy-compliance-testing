
# Policy Compliance Testing

## Project Overview
This project is a Policy Compliance Testing system built using Spring Boot. It allows managing policy data and checking compliance through REST APIs.

## Technologies Used
- Java
- Spring Boot
- Spring Data JPA
- H2 Database
- REST APIs
- Postman
