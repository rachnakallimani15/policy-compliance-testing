# Policy Compliance Testing

## Project Overview
This project is a Policy Compliance Testing system developed using Spring Boot. It allows users to manage and evaluate policy compliance data through REST APIs.

## Features Implemented (Till Now)

### Backend (Spring Boot)
- CRUD Operations:
  - Create Policy (POST)
  - Get All Policies (GET)
  - Get Policy by ID (GET)
  - Update Policy (PUT)
  - Delete Policy (DELETE)

- Dashboard API:
  - Returns total, pass, and fail policy counts
    
## Technologies Used
- Java
- Spring Boot
- Spring Data JPA
- H2 Database (In-Memory)
- REST APIs
- Postman (for testing)
